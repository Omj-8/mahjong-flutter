## 🎯 **プロジェクト概要**

### **アプリ名**: 麻雀ウマオカ計算機
### **技術スタック**:
- **フロントエンド**: Flutter (Dart)
- **プラットフォーム**: Android, Windows, Web
- **データ管理**: SharedPreferences (ローカルストレージ)
- **CI/CD**: GitHub Actions
- **分析**: Firebase Analytics (準備中)
- **ストア**: Google Play Store (申請予定)

### **主要機能**:
1. **麻雀ウマオカ自動計算** (5捨6入方式)
2. **飛びルール完全対応** (ペナルティ・ボーナス設定)
3. **ゲーム履歴・累計管理** (永続化ストレージ)
4. **カスタム設定** (ウマ・オカ・飛び設定)
5. **結果共有機能** (テキストコピー)
